# Assignment for lecture

You are building a command-line text analysis tool that processes a text file and provides statistical insights about its contents.

Your goal is to design and implement a modular Python program that analyzes text data.

Your program must:

Accept a .txt file as input.

Compute:

Total word count

Unique word count

Top 10 most frequent words (excluding stopwords)

Average word length

Longest word(s)

Allow the user to:

Search for a word and see how many times it appears

Handle:

Missing files

Empty files

Non-text input gracefully
